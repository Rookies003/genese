Le Développement Front

Quels sont les types de développement.

  • Le développement Front-End, c’est la partie visuelle d’un site, on va se concentrer sur l’aspect esthétique, les animations, les interactions, le tout pour avoir un rendu qui plaira à l’utilisateur, pour qu’il passe une bonne expérience sur nos sites.
  • Le développement Back-End, c’est le développement côté serveur, c’est là où on gère les bases de données.
  • Le développement Web fait référence à tous les langages qui peuvent entrer dans la création d’un site, il regroupe donc le Front & le Back.
  • Le développement Full-stack fait aussi référence au Front & Back mais on parle généralement de Full-Stack lorsqu’on maîtrise des technologies apparues récemment.

    • Le développement web est vraiment vaste, son écosystème grandit chaque année. Sans plan précis on ne peut pas acquérir de connaissances solides qui permettent de s'épanouir. Ce qu’il faut faire c’est comprendre les bases. Ne pas regarder ce qui brille en premier, mais plutôt regarder les fondations et comprendre comment la maison est construite.

    Pour construire des sites en Front, on a besoin de trois piliers. HTML, CSS, JavaScript. Voilà ce qu’on doit bien comprendre, voilà ce qu’on doit bien maîtriser. Une fois qu’on maîtrise ces trois là, alors toutes les mises à jours, les outils, les « framework », sont faciles à comprendre.
